AECOM - London, UK

Cost.Carbon - Carbon & Cost Estimation Platform

Converted a complex Excel-based carbon and cost estimation tool into a full-stack web application. Streamlined assembly calculations for accurate final cost and carbon footprint reporting across RIBA project stages.

  • Multi-stage input flow covering RIBA Stage I and II with benchmark project comparisons
  • Key Rates panel with editable cost inputs per material - concrete, steel, M&E, facades and more
  • Summary dashboard with elemental breakdown of embodied carbon per m2 (KgCO2/m2)
  • Scenario Comparison Analysis tab for side-by-side project cost and carbon modelling
  • Export Data and Save as PDF output built into the report view
